#Instrucciones de juego


#Objetivo: Adivinar la palabra secreta antes de agotar las vidas.

#Inicio: Al cargar index.php se crea una nueva partida y se muestra la palabra oculta con signos de interrogación.

#Entrada de letras: Usa el campo de texto para introducir una letra y pulsa “Adivinar”.

#Aciertos: Las letras correctas se revelan en su posición.

#Fallos: Cada letra incorrecta resta una vida.

#Letras usadas: Se lista el historial de letras ya probadas.

#Fin de partida:

#Has ganado: Se redirige a ganado.php.

#Has perdido: Se redirige a perdido.php.

#Reiniciar: En ganado.php y perdido.php, pulsa “Jugar de nuevo” para volver a index.php y crear una nueva partida.
